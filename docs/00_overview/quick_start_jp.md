# クイックスタート

## 要件

- Python 3.11+
- Docker
- Prefect 3.x
- uv (パッケージマネージャー)

---

## セットアップ

1. `uv` がインストールされていない場合はインストールします。
2. リポジトリをクローンします。
3. 仮想環境の構築と依存関係のインストールを行います：
   ```bash
   uv sync
   ```
4. 環境変数 (`.env`) を設定します。
5. `config.yaml` を準備します。

---

## 実行フロー

1. Scoutを実行します。
2. サウンドトラックファイルを ingest ストレージにアップロードします。
3. Prefectフローをトリガーします。
4. ワーカーがファイルを処理します。
5. archive/review の結果を確認します。

---

## 注意事項

- Dockerが動作していることを確認してください。
- Prefect APIにアクセスできることを確認してください。
- ストレージのエンドポイントに到達できることを確認してください。
