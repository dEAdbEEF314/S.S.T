# UI Specification: S.S.T Web Interface

## Architecture
- **Backend/BFF**: FastAPI (Python)
  - Acts as a proxy to S3 (SeaweedFS).
  - Handles aggregation of tracks by Album Name instead of AppID.
  - Generates ZIP files on-the-fly or caches them.
- **Frontend**: Vue.js or FastHTML (Lightweight, fast development).
- **Deployment**: Runs as a new `ui` container orchestrated via `docker-compose`.

## State Management & Safety Rule
**CRITICAL**: Incomplete albums MUST NOT be displayed.
An album is considered "Ready" (Complete) ONLY IF:
1. The total number of processed tracks in `archive/` or `review/` exactly matches the `track_count` generated by Scout in the initial JSON.
2. There are no active Prefect tasks running for this AppID.

## Page 1: Archive (Success)
Target: `archive/` directory.

### List View
- Grouped by **Album Name**.
- Columns:
  - Album Name
  - Track Count
  - Formats (e.g., AIFF, MP3)
  - Date Processed
- Actions: 
  - `[+] Expand Details`
  - `[Download ZIP]`

### Detail View (Expanded)
- Table of tracks with applied metadata (Title, Artist, Genre, Year).
- Preview of the embedded 500x500 Cover Art.

## Page 2: Review (Action Required)
Target: `review/` directory.

### Workflow & Purpose
Provides information to assist manual tagging via external tools (e.g., MP3Tag). Users will download the ZIP, tag locally, and re-upload/manage manually.

### List View
- Grouped by **Album Name**.
- Columns:
  - Album Name
  - Track Count
  - **Reason for Review** (e.g., "Metadata Missing", "Cross-Format Mismatch").
- Actions:
  - `[+] Expand Details`
  - `[Download ZIP]`

### Detail View (Expanded) & Helper Info
To assist the user in manual tagging, the UI will display data collected during the system's attempt:
- **Missing Data Highlight**: Table highlighting which fields (Artist, Year, etc.) are empty.
- **Steam Store Data**:
  - `Developer` / `Publisher`
  - `Steam App URL` (Clickable link, opens in new tab).
- **MusicBrainz Data (If available but low confidence)**:
  - `MBID`
  - `VGMdb URL` (Clickable link, opens in new tab).
- **Helper Links Panel**:
  - Direct link to Steam Store Page.
  - Direct link to VGMdb (if found via MusicBrainz).
